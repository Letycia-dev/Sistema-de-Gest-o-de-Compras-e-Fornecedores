# Sistema-de-Gest-o-de-Compras-e-Fornecedores
Aplicação para cadastro e gerenciamento de fornecedores

Aplicação Full Stack desenvolvida com React no front-end e Python no back-end, com o objetivo de gerenciar fornecedores, produtos e pedidos de compra. O sistema permite operações de cadastro, edição, exclusão e listagem de dados, consumindo uma API REST e persistindo informações em banco de dados relacional.
